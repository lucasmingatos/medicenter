# Projeto: Clone de Template — Site Médico (Estudos)
Projeto desenvolvido com finalidade educacional, recriando (clone) um template de site com temática médica para praticar estruturação com HTML, estilização com CSS e interações simples com JavaScript.

## Visão geral
O objetivo deste projeto foi treinar fundamentos de front-end, incluindo:

Marcação semântica com HTML

Estilos básicos com CSS

Responsividade com @media queries

Menu responsivo (mobile) utilizando JavaScript

## Tecnologias utilizadas
HTML5

CSS3 (incluindo @media)

JavaScript (vanilla) para controle do menu responsivo

## Funcionalidades
Layout responsivo para diferentes tamanhos de tela (desktop, tablet e mobile)

Menu de navegação responsivo (abre/fecha em telas menores)

Estrutura de páginas/seções baseada em template de site médico

Estrutura do projeto (exemplo)
Ajuste os nomes conforme o seu repositório.

index.html — Página principal

css/

style.css — Estilos do site + media queries

js/

script.js — Lógica do menu responsivo

assets/ — Imagens/ícones/fontes (se houver)

33Responsividade
A responsividade foi implementada com @media queries, adaptando layout, espaçamentos, tipografia e o comportamento do menu conforme a largura da tela.
